```

그리고 시 단위가 1부터 23까지라면 시 단위에서 하나를 빼주고 만약 시 단위가 0이라면 시를 0으로 초기화 시켜줍니다. 이 조건들을 묶어서 if문으로 걸어주고 만약 분 단위가 45보다 크다면 원래 있던 분에서 45만 빼주면 되기 때문에 else로 나머지 조건을 걸어줍니다.
```
